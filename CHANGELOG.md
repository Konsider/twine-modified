# Changelog

All notable changes in this fork will be documented in this file.

## v1.0.0 — Initial public release

### Added / Changed
- Ctrl + mousewheel zoom on the story map
- Spell check in the passage editor
- Word count per passage
- Default save directory for `.twee`
- Option to make SugarCube the default story format
- Passage right-click menu (copy/paste/make link)
- Tags shown in passage editor bottom bar
- Sort stories by date
- Start maximized window
- Default first passage name “Start”
- Double-click empty map area to create passage
- Drag & drop to create links between passages
- Right-click passage to unlink
- Attach image to passages (editor-only; no export)
- Confirmed ending toggle per passage + shading + summary counts
- Minimap + toolbar info + zoom-to-fit
- Bulk story import/export + improved view/zoom options
- Various bug fixes, UI improvements, and efficiency improvements
