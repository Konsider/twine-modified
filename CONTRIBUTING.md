# Contributing

This is an unofficial fork maintained in spare time.

## Reporting issues
- Please include your OS (Windows/macOS/Linux) and TwineJS Modified version/tag.
- If it’s a crash: include steps to reproduce and any console/log output if available.
- If it’s a UI bug: include a screenshot and what you expected to happen.

## Pull requests
Smaller, focused PRs are easier to review than one huge change. If you’re unsure, open an issue first describing what you want to improve.
